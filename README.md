## Project Overview
This project analyzes the Titanic dataset using Exploratory Data Analysis (EDA) and Data Visualization techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- Female passengers had higher survival rates.
- First-class passengers had better survival chances.
- Passenger class significantly influenced survival.
- Higher fares were associated with improved survival outcomes.
